# FilmFusion 🎬

FilmFusion is a movie ticket booking website designed as a user interface to represent a film booking platform. This project is built using **React** with **Vite** for fast development and **Tailwind CSS** for styling simplicity and responsiveness.

## Features 🌟

- Clean and minimal UI design for movie booking platforms.
- Responsive layout, ensuring compatibility across devices.

## Installation 🚀

1. Clone the repository:
   ```bash
   git clone https://git@github.com:krushna-simform/filmfusion.git
   ```
2. Navigate to the project directory:
   ```bash
   cd filmfusion
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run server
   ```
